# 大闹天宫

话说在那东胜神州傲来国花果山上，一块受天真地秀、日精月华的仙石，突然迸裂，产出一只石猴。这石猴目运金光，射冲斗府，惊动了玉皇大帝。他后来拜师菩提祖师，学得七十二般变化，一个筋斗就是十万八千里，神通广大，自号“齐天大圣”。

一日，大圣在花果山与众猴欢宴，忽见两个独角鬼王来投，并献上一件赭黄袍。大圣大喜，正缺一件像样的行头。鬼王趁势进言道：“大王有如此神通，何不也做个‘齐天大圣’？”大圣闻听，心中欢喜，即刻命人扯起一杆大旗，上书“齐天大圣”四个大字。消息传到天庭，玉帝震怒，即派托塔李天王率哪吒三太子下界擒拿。

李天王点起十万天兵天将，将花果山团团围住。巨灵神挥动宣花斧，一马当先，前来叫阵。大圣冷笑一声，取出如意金箍棒，只轻轻一挡，便震得巨灵神虎口发麻，斧刃都卷了口。哪吒见状，化作三头六臂，手持斩妖剑、砍妖刀等法器，与大圣大战三十回合。大圣眼明手快，拔下一根毫毛，变作自己的替身，真身却绕到哪吒身后，一棒打中他臂膀。哪吒负痛败走，李天王只得收兵回天庭，向玉帝禀报。玉帝无奈，采纳了太白金星的建议，下旨招安，封大圣为“齐天大圣”，并在蟠桃园旁为他修建了齐天大圣府。

大圣得了封号，在天庭整日无所事事，东游西逛，结交各路神仙。玉帝怕他闲中生事，便派他去看管蟠桃园。大圣进了桃园，只见那桃树参天，果实累累。土地神告诉他，这蟠桃分三等：三千年一熟的，吃了成仙了道；六千年一熟的，吃了霞举飞升，长生不老；九千年一熟的，吃了与天地同寿。大圣听得心痒难耐，趁土地神离去，便偷偷摘了几个最大的桃子，饱餐了一顿。自此，他便隔三差五地来偷桃，将看守蟠桃园的职责抛到了九霄云外。

忽一日，王母娘娘要在瑶池举办蟠桃胜会，派了七衣仙女来摘桃。仙女们寻遍桃园，只见大树上的好桃都所剩无几。正疑惑间，大圣从树上跳下，厉声喝问。当他得知王母并未邀请他这位“齐天大圣”赴宴时，顿时勃然大怒，心中那点被招安的虚荣瞬间化为灰烬。他定住七仙女，一个筋斗云来到瑶池。此时宴席尚未开始，桌上已摆满了琼浆玉液、龙肝凤髓。大圣看得眼红，拔下毫毛变作瞌睡虫，迷倒了准备宴席的仙官力士，然后放开肚量，将那些珍馐美酒吃了个酣畅淋漓。他喝得醉醺醺，又踉踉跄跄地闯入了太上老君的兜率宫，看见丹炉旁的五葫芦金丹，也不客气，一股脑儿全倒进嘴里，像吃炒豆一般咽了下去。

酒醒之后，大圣知道自己闯下了弥天大祸，不敢再回齐天大圣府，便一个筋斗翻回了花果山。然而，他偷桃、盗酒、窃丹的恶行很快便败露了。玉帝闻报后龙颜大怒，立刻派遣四大天王协同托塔李天王、哪吒三太子，率领十万天兵天将，布下天罗地网，发誓要擒拿这无法无天的妖猴。

这一场大战，直杀得天昏地暗，日月无光。大圣变作三头六臂，手持三条金箍棒，如纺车般旋转飞舞，独自一人抵挡那十万天兵。他战退了哪吒，打伤了木吒，直杀得九曜星官闭门闭户，四大天王无影无踪。正在酣战之际，观音菩萨举荐了二郎真君前来助战。这二郎神，同样精通八九玄功，变化多端，与大圣棋逢对手。两人斗法，变化万千：大圣变麻雀，二郎便变饿鹰；大圣变小鱼，二郎便变鱼鹰；大圣变水蛇，二郎便变灰鹤。二人从地上打到天上，又从天上打回地上，难分难解。

就在大圣与二郎神以及他的梅山六兄弟缠斗不休时，端坐在凌霄宝殿的玉帝和观音菩萨都看得焦急。太上老君从袖中取出一个亮灼灼的圈子——金刚琢，瞅准时机，朝大圣的头顶砸了下去。大圣正全神贯注地与二郎神搏斗，冷不防被这法宝正中天灵盖，一个踉跄跌倒在地。梅山兄弟一拥而上，用勾刀穿了他的琵琶骨，使他再也无法变化，随后用铁索捆绑，押解回了天庭。

玉帝下令将孙悟空押赴斩妖台，碎尸万段。然而，天兵天将的刀砍斧剁、雷打火烧，都不能伤他分毫。原来大圣在太上老君的丹炉里偷吃了那五葫芦金丹，早已炼就了金刚不坏之躯。老君无奈，只得将他推进了八卦炉中，用文武火煅烧，想炼出他肚里的金丹。谁知那炉子有八个方位，大圣躲在了有风无火的“巽宫”位下，虽然被烟熏红了眼睛（后来成了火眼金睛），却始终没有被烧死。

七七四十九天期满，老君开炉取丹。大圣纵身一跃，从丹炉中跳出，一脚踢翻了炉子，带着一身火光冲了出来。他抡起金箍棒，一路狂扫，直打得九曜星官魂飞魄散，四大天王望风而逃。他口中高喊：“皇帝轮流做，明年到我家！”竟一路杀向了凌霄宝殿。他势如破竹，无人能挡，那些天兵天将、神仙星宿，都成了他棒下的败将。大圣一路杀到通明殿里，凌霄殿外，幸得佑圣真君的佐使王灵官拼死抵挡，才勉强将他拦住。

这一番惊天动地的打斗，惊动了远在西天的如来佛祖。玉帝连忙下旨，请如来前来降妖。如来赶到后，并没有直接与大圣动手，而是用轻描淡写的语气与大圣打了一个赌：若他能一个筋斗翻出如来的右手掌，便请玉帝将天宫让给他；若翻不出，便再修炼几劫，不要妄自争强。大圣自恃筋斗云天下无双，欣然答应。他纵身一跃，飘飘荡荡，连翻筋斗，只见前方有五根肉红色的柱子，便以为到了天尽头。他得意忘形，拔下毫毛，在柱子上写下“齐天大圣到此一游”八个大字，还撒了一泡猴尿以作凭证。

待他翻回原地，嘲笑如来爽约时，如来却笑道：“你这泼猴，未曾出我手掌！”大圣低头一看，只见如来右手中指上，正写着那八个大字，且指缝间还有一股猴尿的骚气。大圣惊得目瞪口呆，连声叫道：“有这等事！有这等事！俺老孙明明到了天尽头，定是你使了幻术！”他正要再使神通，如来却不容分说，将手掌一翻，化作五行大山，将大圣压在了山下。

五行山上有如来佛祖的封帖，贴上写有“唵、嘛、呢、叭、咪、吽”六个金字，任凭大圣有万般神通，也再难逃脱。从此，齐天大圣便被压在这五行山下，饥餐铁丸，渴饮铜汁，独自一人，度过了五百年漫长的岁月，直到五百年后，东土大唐来了一位西天取经的僧人，这才有了日后师徒相逢的缘法。
